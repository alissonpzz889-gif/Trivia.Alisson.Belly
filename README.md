<Projeto: Trivia – Quiz de Desenhos Animados

Bem-vindo ao Trivia Animada, um quiz simples e divertido criado em Python como atividade da disciplina.
O objetivo do projeto é apresentar um jogo de perguntas e respostas baseado em personagens de desenhos animados, seguindo os requisitos propostos:

✔️ Seleção de dificuldade
✔️ No mínimo 7 perguntas
✔️ Opções de resposta
✔️ Verificação de acertos e erros
✔️ Placar final
✔️ Versionado com Git/GitHub
✔️ Desenvolvido em Python

<Objetivo do Projeto

Este projeto foi criado para treinar conceitos essenciais de programação, como:

Entrada e saída de dados

Estruturas de decisão (if, else)

Listas e dicionários

Laços de repetição

Funções simples

Versionamento com Git e GitHub

Organização de projeto no VSCode

Além disso, o quiz tem como tema desenhos animados, trazendo perguntas leves e divertidas.

<Tema das Perguntas 

As perguntas são baseadas em personagens de:

Bob Esponja

Scooby-Doo

Ben 10

Pokémon

Os Simpsons

Tom & Jerry

E outros clássicos
